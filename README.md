# Umang

The online educational guidance system, developed with HTML, CSS, JavaScript, PHP, and MySQL, helps users make informed educational choices. It offers course selection assistance, tailored internship opportunities, entrance exam details with syllabi, and practice links for placement prep. Additionally, users can access previous years' exam papers via QR codes, providing a comprehensive tool for academic planning.
